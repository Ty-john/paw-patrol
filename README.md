# Paw Patrol Website
## Project Overview
This website was developed by BSIT 3rd year students as part of their Web Development course. It
showcases the services, mission, and contact information of Paw Patrol, a fictional pet-driven initiative company.

## Purpose
To provide an online presence for Pet Care initiative, promote its services, and allow users to contact the
team directly.

## Design Concept
- **Color Scheme**: Purple, white, and orange for pet-branding
- **Typography**: Sans-serif fonts for readability and LEMON for headers
- **Layout**: Consistent spacing, mobile-friendly grid
